## Запуск программы:

1. Склонируйте репазиторий к себе на компьтор.
- git clone https://github.com/KirillKovaluk/AI-for-CAD-model.git

2. Перейдите в папку с программой
-  AI-for-CAD-model

3. Запустите программу.
- CadGeneratorWinForms.sln

4. В вводную строку напишите какую фигуру вам хочется получить(базовые).
- Пример: нарисуй домик

5. Выберите цвет.
- Представленные цвета в списке

6. Откройте полученный файл в 3D-Paint.
